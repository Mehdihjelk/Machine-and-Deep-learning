# ML & DL Algorithms From Scratch

A comprehensive collection of core Machine Learning and Deep Learning algorithms implemented from scratch. This repository serves as a personal sandbox to bridge the gap between theoretical mathematics and practical, foundational code.

Instead of relying heavily on black-box libraries, the models here are built from the ground up to deeply understand their underlying mechanics.

## Project Architecture

To keep the codebase clean and modular, the project is divided into source code and an experimentation playground:

```text
📦 project-root
 ┣ 📂 src/                  # Contains the algorithm implementations
 ┃ ┣ 📜 linear_regression.py
 ┃ ┣ 📜 logistic_regression.py
 ┃ ┗ 📜 ... (other algorithms)
 ┣ 📓 main.ipynb      # The main notebook to train and test the models
 ┣ 📜 requirements.txt      # Project dependencies
 ┗ 📜 README.md
